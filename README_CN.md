# Disc Jockey：Next

一款 Fabric 模组，可自动与游戏内音符盒交互，播放 [Note Block Studio](https://www.stuffbydavid.com/mcnbs) 制作的 `.nbs` 格式音乐。

> 基于 [Disc Jockey](https://github.com/SemmieDev/Disc-Jockey) 分支，进行了大量功能增强、界面重设计和 Bug 修复。

---

## 相比原版新增功能

### 播放核心
- **4 种循环模式** — 不循环、列表循环、单曲循环、**随机循环**（每次随机切换下一首）
- **倍速调节** — 通过界面按钮以 0.25x 步长在 0.10x 到 10.00x 之间调节播放速度
- **暂停恢复保护** — 自动检测游戏暂停/恢复，防止取消暂停时音符堆积一次性爆发
- **播完自动停止** — 循环模式设为"不循环"时，歌曲播完自动停止

### 歌曲管理
- **收藏夹系统** — 点击星标收藏/取消收藏，独立收藏夹视图快速访问
- **文件夹浏览** — 支持子目录整理歌曲，GUI 内可进入文件夹、返回上级
- **三种排序方式** — 按名称、创建时间、最近播放排序，每种均可切换正序/倒序
- **实时搜索** — 输入即过滤，忽略空格，匹配歌曲名和文件名
- **重新加载** — 一键从磁盘重新加载所有歌曲，无需重启
- **双击播放** — 双击歌曲列表中的任意歌曲立即开始播放
- **拖拽导入** — 将 `.nbs` 文件直接拖入模组界面即可导入
- **打开音乐文件夹** — 一键在文件资源管理器中打开歌曲目录

### 界面交互
- **进度条跳转** — 点击进度条任意位置跳转播放进度，实时显示 `当前时间 / 总时长`
- **精准跳转输入框** — 输入精确时间戳（如 `1:30` 或 `90`），支持 `MM:SS` 格式
- **悬停提示** — 鼠标悬停任意按钮显示功能说明
- **所需方块叠加层** — 右上角 HUD 面板，显示当前歌曲需要的音符盒乐器类型及附近已有的数量，按行颜色区分（绿色=充足，红色=不足），并显示方块名称
- **自定义设置界面** — 完全不依赖 Cloth Config GUI，自主实现；支持滚动、直接输入数值、+/- 按钮、保存配置和恢复默认
- **圆角设计** — 方块面板和工具提示均采用适量圆角

### 体验优化
- **创造模式提醒** — 在创造模式下点击播放会弹出提示，告知需要切换到生存模式
- **播放时自动关闭方块面板** — 可选，开始播放时自动隐藏所需方块提示框

## 安装方法

1. 安装 [Fabric Loader](https://fabricmc.net/use/)、[Fabric API](https://modrinth.com/mod/fabric-api) 和 [Cloth Config API](https://modrinth.com/mod/cloth-config)
2. 从 [Releases](https://github.com/xin-build/Disc-Jockey-Next/releases) 下载最新 `.jar`
3. 放入 `.minecraft/mods/` 文件夹
4. 将 `.nbs` 歌曲文件放入 `.minecraft/config/disc_jockey/songs/`（支持子目录）
5. 启动游戏，按 `J` 打开菜单

---

## 使用方法

1. **在周围放置对应的音符盒**（可查看方块提示面板确认所需乐器）
2. 按 `J` 打开 Disc Jockey 界面
3. 选择歌曲，设定循环模式和速度
4. 点击 **播放** — 模组将自动调音周围音符盒并开始演奏
5. 使用**进度条**跳转进度、**速度按钮**调节倍速、**跳转输入框**精确定位

## 配置项

通过歌曲界面的扳手图标或 Mod Menu 进入设置：

| 设置项                 | 默认值 | 说明 |
|------------------------|--------|------|
| 隐藏警告               | 关闭   | 隐藏反作弊提醒 |
| 禁用异步播放           | 关闭   | 在主线程同步播放（可解决部分服务器播放失效问题） |
| 全向音符盒音效         | 开启   | 播放时所有音效来自全方向，更悦耳 |
| 延迟播放               | 0.0 秒 | 调音完成后延迟指定秒数再开始播放 |
| HUD X / Y 坐标         | 2, 2   | 方块提示面板位置 |
| HUD 宽度               | 180    | 方块提示面板宽度（像素） |
| 默认速度               | 1.0    | 默认播放速度 |
| 播放时关闭提示框       | 开启   | 开始播放时自动隐藏方块提示面板 |
